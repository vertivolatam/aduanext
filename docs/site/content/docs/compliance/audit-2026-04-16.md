### P0.5 — RBAC + tenant isolation — DONE

**Estado:** End-to-end completo en 4 layers.

1. Domain entities (VRTV-55): User, Tenant, Role, TenantMembership
2. `AuthorizationPort` + `InMemoryAuthorizationAdapter`
3. `KeycloakAuthorizationAdapter` con JWT + JWKS cache (VRTV-60)
4. apps/server middleware + role guards + route table (VRTV-61)
5. Postgres RLS en audit_events + legal_holds (VRTV-62, 73)

**PRs:** #16, #19, #20, #21 (VRTV-55, 60, 61, 62)
